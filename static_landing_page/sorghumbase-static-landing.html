Getting access to the Sorghumbase CMS

Much of the data displayed on the Sorghumbase web site is entered in WordPress, a popular blogging system. WordPress has grown to become a full content management system (CMS), which means that you can enter data like posts, blogs, and other web content in an easy to use interface that does not require any programming knowledge.

You will need to have a WordPress login to enter new content for Sorghumbase. If you have not already been given a login, please send an email to contact[at]sorghumbase[dot]org to request one.

About Us

Sorghumbase aims to be a portal that enables researchers and growers of sorghum to exploit data management resources, relevant biological knowledge and genetic resources in a manner that effectively addresses the needs of the sorghum community to increase our ability to understand complex biological traits that are critical to U.S. agriculture.
